// About the engine

What is YASA?

YASA (Yet Another Spindle Algorithm) is an open-source Python library for automated analysis of polysomnographic recordings, developed by Raphael Vallat. It delivers state-of-the-art, machine-learning–powered sleep staging directly from raw EEG signals.

  • 🧠
    Automated sleep staging — classifies 30-second epochs into Wake, N1, N2, N3, and REM using a gradient-boosting classifier trained on thousands of PSG recordings.
  • 〰️
    Spindle & slow-wave detection — detects sleep spindles and slow oscillations with configurable thresholds, returning per-event statistics.
  • 👁️
    REM detection — identifies REM epochs and computes REM-related metrics including NREM/REM transitions.
  • 📊
    Sleep statistics — computes total sleep time, sleep efficiency, latencies, WASO, and full stage percentages per AASM standards.
  • 📡
    Bandpower analysis — spectral power in delta, theta, alpha, sigma, and beta bands per epoch and per channel.
# Core YASA pipeline import yasa, mne raw = mne.io.read_raw_edf("recording.edf") sls = yasa.SleepStaging(raw, eeg_name="C4-M1") hypno = sls.predict() stats = yasa.sleep_statistics(hypno, sf_hyp=1/30)
// The web application

What is YASAFlaskified?

YASAFlaskified wraps the YASA engine in a production-grade Flask application, letting clinicians and researchers run full sleep analyses through a browser — no Python knowledge required. Jobs run asynchronously, results are stored, and reports are delivered in seconds.

📁

EDF Upload

Drag-and-drop or select any EDF file from your PSG system. The app validates the recording, extracts available channels, and queues the job immediately.

⚡

Async Processing

Powered by Redis Queue (RQ) and background workers, heavy analyses run without blocking the UI. Real-time job status keeps you informed throughout.

📈

Rich Reports

Hypnograms, sleep statistics tables, spindle maps, slow-wave detections, and bandpower charts — all rendered server-side with Matplotlib and downloadable as PNG/CSV.

🔒

Secure & Hardened

CSRF protection, rate limiting, strong session management, UFW firewall, and Fail2ban — built for clinical environments where data security is non-negotiable.

🐳

Docker Deployment

Fully containerised with Gunicorn + Nginx reverse proxy. One-command deploy on any Linux server, with Nginx Proxy Manager for multi-domain SSL setups.

🧪

Research-Grade

Supports AASM-compliant staging, configurable epoch length, multi-channel selection, and raw µV signal preprocessing — trusted by sleep researchers.

// How it works

From EDF to insights in four steps

01

Upload EDF

Select your polysomnography recording. The app accepts standard European Data Format files from any PSG device.

02

Configure

Choose your EEG, EOG, and EMG channels by name. Set epoch length and select the analyses you need.

03

Analyse

YASA's gradient-boosting model stages every epoch. Spindle, slow-wave, and bandpower detectors run in the background.

04

Download Report

Receive your hypnogram, statistics table, and event maps. Export to PNG and CSV for further analysis or clinical records.
