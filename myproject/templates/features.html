App Features

Welcome to our EEG Sleep Staging Application! Below is a summary of the app's capabilities:

1. User Authentication

  • Secure login/logout for registered users.
  • New user registration with password encryption.

2. File Upload

  • Supports uploading multiple `.edf` EEG files.
  • Ensures only valid files are processed.

3. Background Processing

  • Handles long-running EEG file processing tasks using Redis-backed RQ workers.
  • Processes EEG data to generate hypnograms (PDF) and sleep stage data (CSV).

4. Real-Time Job Monitoring

  • Tracks job progress on the "Processing" page.
  • Automatically redirects to results upon completion.

5. Results Display and Download

  • Displays processed files with links to download PDF and CSV results.
  • Organized storage of results in the `processed/` folder.

6. Admin and Logging

  • Automatically creates an admin account for database initialization.
  • Logs all actions for auditing and debugging purposes.

Future Possibilities

The app is designed for scalability and can support additional features such as visualizations, extended file format support, and more advanced analytics.
